# ideas

1. Add a side panel showing dependency graph of variables/functions so users can see what each row depends on.
2. Support row grouping/sections (for example "Inputs", "Calculations", "Checks") with collapsible headings.
3. Add undo/redo history for edits, drag operations, and row add/remove actions.
4. Introduce row pinning/locking to prevent accidental drag or edits on validated rows.
5. Add import/export for worksheets as JSON and CSV (in addition to localStorage saves).
6. Build a unit autocomplete dropdown and inline dimensional mismatch warnings while typing.
7. Add function library snippets (beam bending, voltage divider, ideal gas, Reynolds number, etc.) insertable as templates.
8. Add comments/annotations per row so assumptions and references can be documented next to calculations.
9. Provide a "report mode" that renders a clean print/PDF layout with ordered equations and results.
10. Add optional collaboration mode using a backend (shared worksheets + version snapshots).
